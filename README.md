# NovaSoftSolutionsSL_EP

MyPersonalShopper – Proyecto de Ingeniería del Software (UAB 2025-2026)
Descripción general

MyPersonalShopper es una plataforma de Personal Shopper online que ofrece un servicio de asesoramiento personalizado en moda. Los usuarios crean un perfil con su talla, gustos y preferencias, y un personal shopper asignado selecciona cinco prendas que se ajustan a su estilo.
El cliente recibe el pedido en casa, decide qué prendas conservar y devuelve las demás sin coste adicional.

Este proyecto se desarrolla como proyecto final de semestre de la asignatura Enginyeria del Programari (UAB 2025-2026).
El equipo actúa como una empresa de software contratada por los emprendedores Julieta Serrano, Luis Lara y su equipo para diseñar e implementar el sistema que hará posible la idea de negocio.
